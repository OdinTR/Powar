How To Play Powar
- Go to this directory (Powar\Powar\Powar.exe) and run Powar.exe
- Click Tutorial button to learn the instructions of the game
- Click Start button to start the game
- Each player tries to push the other out of the icy platform, the player who successfuly pushed the other out is granted with a point
- Each player starts with 0 points and aims to reach to 3 points, the player who reaches 3 points wins the game
- Click Exit button to exit the game

Software made with Unity 5.6.1f1

You can also reach to additional details about the game from this link -> https://github.com/steveberkhan/Powar

Thank you for playing!
